问题 hexo报highlight相关错误

hexo 突然不跑动了，报错如下

改动 dependabot推送依赖升级

昨天 dependabot 推送 pull request 依赖升级

修复过程

按关键词highlight报错搜索，发现我的主题有类型github issue反馈，并被修复关闭了。

遂升级主题，一顿操作下来，折腾了一晚上，还是相同报错，都没睡好觉；

于是只能本地git还原修改，重新切个新工程，并用新主题，排查是否主题原因：

发现没报错。

那就只能是dependabot依赖升级的原因了。

修复方法

  1. 将依赖恢复为原来版本
  2. 禁用dependabot.yml（删除该文件）
    本版手动维护和更新依赖
